Unity FPS Sample

First Person Shooter (FPS) Sample is a open sourced Unity project which has two levels. A test level (level 00) and a game level (level 01) with detailed game assets and environmental effects. Project can be found at: https://github.com/Unity-Technologies/FPSSample

Node-RED Flow

3D FPS Sample is a first-person shooter game provided by unity. The game has been tweaked to work with the REDify library, to demonstrate the flexibility of the REDify library. The game has a similar objective to classic FPS games, the objective is to get as many kills as possible within the set time frame. Each key action within the game, such as shooting, reloading and dying can be assigned to a particular output using an Amazon Echo dot or Philips Hue device. The Philips Hue provides audio commentary to assist the player throughout the game and confirm that a key action has taken place. This provides an immersive and unique gaming experience for games enthusiasts alike!
